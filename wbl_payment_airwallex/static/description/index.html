AirWallex Payment Provider

The AirWallex Payment Method with Refund app for Odoo enables seamless integration with the AirWallex payment gateway. This module supports multiple currencies, allowing businesses to accept payments from global customers. It also includes a refund functionality, enabling admins to process partial or full refunds by simply entering the refund amount.

Work Flow of AirWallex Payment Provider:

Detailed Features and Steps:

  • AirWallex Integration: Direct integration with the AirWallex payment gateway for efficient payment processing.
  • Multi-Currency Support: Accept payments in various currencies, catering to international clients.
  • Refund Functionality: Admins can issue both partial and full refunds to customers.
  • Easy Refund Entry: Admins can quickly enter the refund amount for straightforward transaction adjustments.
  • User-Friendly Interface: Streamlined interface for managing payments and refunds efficiently.
